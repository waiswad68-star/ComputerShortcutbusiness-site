Popular Shortcuts

  • Ctrl + C : Copy
  • Ctrl + V : Paste
  • Ctrl + Z : Undo
  • Ctrl + S : Save

Tips & Tricks

  • Use Alt + Tab to quickly switch between windows.
  • Press Windows + L to lock your computer fast.
  • Use Ctrl + Shift + T to reopen the last closed browser tab.

Examples

Try copying some text (Ctrl+C) and pasting it somewhere else (Ctrl+V).
